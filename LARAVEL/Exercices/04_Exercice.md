# Elixir 

Installez les dépendances suivantes elles serviront à construire la partie front, css, js du projet dans Laravel.

```bash
node -v

# dispo sur la machine
npm install gulp --global

# en local
npm install gulp

# Sous MAC
npm install

# Sous Windows et VM
npm install --no-bin-links

# lance le build
gulp

# pour la production
gulp --production
```
